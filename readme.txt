== Description ==

https://www.youtube.com/watch?v=C9PRsCkI6Fw

Easily track all activity on your WordPress site and notify upon activity detection. Tracks login attempts, registrations, comment actions, post/page actions and much more...

= What can be tracked? =

= User Actions =

* User Registration
* Password Reset Request
* Password Reset Action
* Profile Update
* User Failed Login
* User Successful Login
* Delete User Action
* Change User Role

= File Editor Actions =

* Changes made using WP File Editor

= Media =

* Adding Media
* Editing Media
* Deleting Media
* Saving Media

= Installer =

* Activate Plugins
* Deactivate Plugins
* Theme Change
* Plugin Update
* Theme Update
* WP Core Update

= Comments =

* New Comment
* Edit Comment
* Delete Comment
* Trash Comment
* Untrash Comment
* Spam Comment
* Duplicate Comment

= Settings =

* Update WordPress Options
* Update Site Options
* Transient Updates

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Setup Notifications settings in WP Admin "WP Activity Monitor".
4. Enjoy!
